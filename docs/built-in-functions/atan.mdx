## 説明

`tan`は角度を与えると対応する正接値を返す一方で、`atan`は正接値を与えると対応する角度を返す。

<LatexPlot xMin={-5} xMax={5} yMin={-Math.PI/2} yMax={Math.PI/2} samples={600} plotFunction={(x) => Math.atan(x)}>
$$
y = \arctan(x)
$$
</LatexPlot>

x, y, z軸にそって回転する際に使う。らしい。
