## 説明

引数に渡された値をθとして、`tan(θ)`の値を返す。

`sin`や`cos`と違って連続的ではないグラフが特殊で、使い所がよく分からない。`θ`が`π/4`の時、`1`となる。

<LatexPlot xMin={-3.1} xMax={3.1} yMin={-5} yMax={5} samples={700}
    plotFunction={(x) => {
        const c = Math.cos(x);
        // cos(x) が 0 付近では非連続のため、線を切る
        if (Math.abs(c) < 0.02) return NaN;
        return Math.sin(x) / c;
    }}
>
$$
y = \tan(x)
$$
</LatexPlot>
