## 説明

`cosh`は`x`における双曲線余弦を返す一方で、`acosh`は`x`における逆双曲線余弦を返します。つまり、`cosh`の逆関数です。

<LatexPlot xMin={1} xMax={5} samples={500} plotFunction={(x) => Math.acosh(x)}>
$$
y = \operatorname{acosh}(x)
$$
</LatexPlot>
